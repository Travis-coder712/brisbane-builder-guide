# Brisbane Builder Guide

A practical guide PWA for selecting, vetting and budgeting a Brisbane home builder for renovations.

**Live at:** https://travis-coder712.github.io/brisbane-builder-guide/

## Stack

React 19 · TypeScript · Vite 6 · vite-plugin-pwa

## Local development

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # produces ./dist
```

## Deploy

Push to `main` — GitHub Actions builds and publishes to GitHub Pages automatically.

## History

Extracted from `Travis-coder712/aures-db` (where it previously lived at `/aures-db/builders/`) into its own repo + deployment in May 2026, to keep concerns separated.
